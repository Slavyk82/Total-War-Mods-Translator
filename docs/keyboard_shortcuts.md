## Tips for Efficiency

### Common Workflows

**Quick Translation Workflow**:
1. `Ctrl+3` - Open Projects
2. `Enter` - Open selected project
3. `Ctrl+A` - Select all entries
4. `Ctrl+T` - Translate selected
5. `Ctrl+S` - Save changes

**Search and Edit Workflow**:
1. `Ctrl+F` - Open search
2. Type search term
3. `Ctrl+Enter` - Execute search
4. `F2` - Edit first result
5. Make changes
6. `F3` - Find next
7. Repeat

**Review and Validate Workflow**:
1. Filter to show untranslated: `Ctrl+Shift+F`
2. Navigate with `Arrow Keys`
3. Edit with `F2`
4. Mark validated: `Ctrl+M`
5. Next entry: `Arrow Down`

### Customizing Shortcuts

> **Note**: TWMT does not currently support custom keyboard shortcuts. This feature is planned for a future release.

## Accessibility

### Keyboard Navigation

All features in TWMT are accessible via keyboard:
- Use `Tab` to move between controls
- Use `Arrow Keys` to navigate lists and grids
- Use `Enter` or `Space` to activate buttons
- Use `Esc` to cancel actions

### Screen Reader Support

TWMT is compatible with Windows Narrator and other screen readers:
- All controls have proper labels
- Focus indicators are visible
- Navigation order is logical

### High Contrast Mode

TWMT respects Windows High Contrast settings:
- Press `Left Alt+Left Shift+Print Screen` to toggle (Windows)
- TWMT UI adapts automatically

## Shortcut Conflicts

If you have other applications running, some shortcuts may conflict:

**Common Conflicts**:
- `Ctrl+Shift+T`: Browser "Reopen Tab"
- `Ctrl+N`: Browser "New Window"
- `Ctrl+W`: Browser "Close Tab"

**Solution**: Close conflicting applications or use alternative shortcuts where available.
