# TWMT Keyboard Shortcuts

**Version 1.1** | **Last Updated**: December 2025

Reference for all keyboard shortcuts currently implemented in TWMT.

---

## Global Navigation

Use these shortcuts from anywhere in the application.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+1` | Home | Go to Home dashboard |
| `Ctrl+2` | Mods | Go to Mods browser |
| `Ctrl+3` | Projects | Go to Projects list |
| `Ctrl+4` | Glossary | Go to Glossary management |
| `Ctrl+5` | Translation Memory | Go to Translation Memory browser |
| `Ctrl+6` | Settings | Open Settings |
| `Ctrl+Home` | Home | Go to Home dashboard (alternative) |

> **Note**: Navigation shortcuts are blocked while translation or pack generation is in progress.

---

## Translation Editor

These shortcuts work in the Translation Editor screen.

### Editing & Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last translation change |
| `Ctrl+Y` | Redo | Redo last undone change |
| `Ctrl+Shift+Z` | Redo | Redo last undone change (alternative) |
| `Ctrl+C` | Copy | Copy selected entries to clipboard (TSV format) |
| `Ctrl+V` | Paste | Paste from clipboard and update translations |
| `Delete` | Delete | Delete selected translations (with confirmation) |
| `Escape` | Clear Selection | Clear all row selections |

### DataGrid Built-in Navigation

These shortcuts are provided by the Syncfusion DataGrid component:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Arrow Keys` | Navigate Cells | Move between cells in DataGrid |
| `Tab` | Next Cell | Move to next editable cell |
| `Shift+Tab` | Previous Cell | Move to previous editable cell |
| `Enter` | Edit Cell | Start editing selected cell |
| `F2` | Edit Cell | Start editing selected cell (alternative) |
| `Escape` | Cancel Edit | Cancel current edit and revert changes |
| `Page Up/Down` | Scroll | Scroll one page up or down |
| `Click` | Select Row | Select single row |
| `Ctrl+Click` | Toggle Selection | Add/remove row from selection |
| `Shift+Click` | Range Selection | Select range from last selected to clicked row |

## Accessibility

### Keyboard Navigation

- Use `Tab` to move between controls
- Use `Arrow Keys` to navigate lists and grids
- Use `Enter` or `Space` to activate buttons
- Use `Escape` to cancel actions

### Screen Reader Support

TWMT is compatible with Windows Narrator and other screen readers:
- All controls have proper labels
- Focus indicators are visible
- Navigation order is logical

### High Contrast Mode

TWMT respects Windows High Contrast settings:
- Press `Left Alt+Left Shift+Print Screen` to toggle (Windows)
- TWMT UI adapts automatically
