## What's New - 1.2.2

### New Features

-Added automatic image generation for translation packs: the pack image is now generated using the translated mod's image with the target language flag overlaid. Existing pack images are preserved and won't be overwritten.
-Release notes are now displayed after each application update

### Improvements

-Source text can now be selected in the translation editor for easy copying
-Double-click on a field in the translation editor to manually edit a translation unit
