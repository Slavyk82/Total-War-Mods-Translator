## What's New — v2.0 (Major Update)

### Highlights
- The app is now available in multiple languages: English, French, German, Spanish, Chinese, Russian, and Korean
- Complete UI overhaul for a streamlined, faster workflow
- Brand-new "Home" screen that surfaces, at a glance, every translation that needs updating after an original mod has been updated
- New publishing screen to push your projects to the Steam Workshop directly from TWMT
- The sidebar now reflects the workflow stages, so you always know where you are

### Workflow & Editor
- Fully reworked and simplified end-to-end workflow
- Clickable workflow cards and actions on the Home screen
- Clearer step-by-step workflow inside the translation editor
- Batch processing for projects on the "Projects" screen (button in the top-right corner)

### LLM & Translation
- Updated LLM integrations: latest models available for OpenAI, Anthropic, and DeepSeek
- Unified glossaries: one glossary per game and per language
- Improved Translation Memory (TM) handling and performance
- Improved full-game translation into a language not officially supported by the base game

### Tools
- New conflict checker in the translation compilation builder
- Theme selector added (bottom-left of the window)

### Performance & Reliability
- Significantly improved database responsiveness across the app
- Better error handling and clearer error reporting
